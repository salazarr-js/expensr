# @slzr/expensr-api

Hono API server running on Cloudflare Workers. Pure route library that exports `app` — imported by `packages/web/server/index.ts` for production deployment.

Scaffolded from [cloudflare-hono template](https://github.com/cloudflare/workers-sdk/tree/main/packages/create-cloudflare/templates/hono/workers/templates).

## Commands

```bash
pnpm dev          # local dev via wrangler (standalone)
pnpm deploy:cf    # deploy to Cloudflare Workers
pnpm typegen:cf   # regenerate types from wrangler.jsonc
```

## Bindings

`CloudflareBindings` is auto-generated by `pnpm cf-typegen` into `worker-configuration.d.ts`. Pass it as a generic to Hono:

```ts
const app = new Hono<{ Bindings: CloudflareBindings }>()

app.get("/example", (c) => {
  // c.env.DB, c.env.KV, etc. — fully typed
})
```

Add bindings in `wrangler.jsonc`, then run `pnpm typegen:cf` to update the types.
